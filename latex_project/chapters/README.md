Each approved chapter will live here as its own `.tex` file (e.g.,
`01_introduction.tex`) and be included from `../main.tex` via `\input{}`.

`../../content/markdown_drafts/`. Do not author chapters here from scratch
without first going through the Markdown-first review workflow.
