# content/markdown_drafts/

> **Status:** scaffold only — no draft chapters exist yet.

Article drafts are written and reviewed here in Markdown first because
Markdown is far easier to inspect and edit than raw LaTeX. Drafts will:

- One chapter or section per `.md` file (naming convention TBD).
- Use real, cited content backed by entries in
  `../../latex_project/references.bib`.
- Be approved by a human reviewer before LaTeX conversion.

Do not commit fabricated content, fabricated sources, or speculative material
to this directory.
